单链表的结点删除

发表于 2016-11-10   |   分类于 数据结构——链表   |  

从最基本的写起，今天写的是单链表的某个结点的删除；
节点的删除分为两种情况，分别是该节点不为尾结点，第二种是该节点为任意结点（有可能为尾结点）。

链表结点类的定义

Definition for singly-linked list.
public class ListNode {
    int val;
    ListNode next;
    ListNode(int x) { val = x; }
}

非尾结点的删除

//删除非最后一个结点
    public static void deleteNode1(ListNode p){
        if(p==null||p.next==null) return;
        ListNode pNext=p.next;
        p.val=pNext.val;
        p.next=pNext.next;
    }

删除任意一个结点

//删除单链表中的任意一个结点,其中L为整个链表
    public static void deleteNode2(ListNode L,ListNode p){
        if(p==null) return;
        if(p.next!=null){
            ListNode pNext=p.next;
            p.val=pNext.val;
            p.next=pNext.next;
        }else{
            ListNode pre=L;
            while(pre.next!=null){
                if(pre.next==p){
                    pre.next=p.next;
                }
            }
        }
    }
