动态规划——背包问题

发表于 2016-11-02   |   分类于 Java   |  

问题描述

假定背包的最大容量为W，N件物品，每件物品都有自己的价值和重量，将物品放入背包中使得背包内物品的总价值最大。

示例：
背包总容量 W = 10
物品数量 N = 4
物品价值 v[] = {10, 40, 30, 50}
物品重量 w[] = {5, 4, 6, 3}

思路

著名的背包问题有两种解法，分别是动态规划和贪婪算法。本文主要使用动态规划。
在现实生活中，有一类活动的过程，由于它的特殊性，可将过程分成若干个互相联系的阶段，在它的每一阶段都需要作出决策，从而使整个过程达到最好的活动效果。当然，各个阶段决策的选取不是任意确定的，它依赖于当前面临的状态，又影响以后的发展，当各个阶段决策确定后，就组成一个决策序列，因而也就确定了整个过程的一条活动路线,
这种把一个问题看作是一个前后关联具有链状结构的多阶段过程就称为多阶段决策过程，这种问题就称为多阶段决策问题。

背包问题可以简化为将i个物品放入容量为w的背包中，求其物品组合中最大的价值；
我们知道往一个以背包中加入一个物体是最简单的，i个物品放入容量为w的背包中可以简化将一个物品放入已经放入（i-1）个物的背包中；
由此递推，我们可以先放入一个物品，然后求出最大值放入二维数组中，然后以放入一个物品为初始状态，继续放入一个物品，依次放入i个物品，由此求出最大值。

运行结果图如下：

对于上图：

1.第一行物品为0，所以value全都为0；
2.第二行只有一个物品，当前容量weight为5时，value值为10，因为后面的capacity均大于5，故后面value值均为10；
3.第三行中，当前容量weight为4时，value值为40，当前容量weight为5时，只能分别装入物体1或物体2，故取其中较大值40；当前容量weight为9时，能同时装下物体1和2，此时value值为50；
4.第四行中，在允许加入第三个物体的条件下，当容量小于等于9时，value值均没有上一行的大，故取上一行的值，当容量为10时，放入物品2和3可以取到最大值70；
5.第五行中，按照以上步骤取得最大值90，并放入最后一个数组元素中。

代码实现

package Algorithms;
public class DynamicProgramming {
    public static int knapsack(int[] value, int[] weight, int capacity)
    {
        int rows1=weight.length;
        int[][] v=new int[rows1+1][capacity+1];
        //因为第一行没有物体，所以价值全部设为0
        for(int col=0;col<=capacity;col++)
        {
            v[0][col]=0;
        }
        //同理第一列的capacity为0，所以第一列全部设为0
        for(int row=0;row<=rows1;row++)
        {
            v[row][0]=0;
        }
        //计算capacity计算最大价值
        for(int item=1;item<=rows1;item++)
        {
            for(int wegt=1;wegt<=capacity;wegt++)
            {    
                if(weight[item-1]<=wegt){//当单个物体重量小于等于capacity时
                    v[item][wegt]=Math.max(value[item-1]+v[item-1][wegt-weight[item-1]],v[item-1][wegt]);
                }else{//当单个物体重量大于capacity时
                    v[item][wegt]=v[item-1][wegt];
                }
            }
        }
        //打印出矩阵
         for (int[] rows : v) {
            for (int col :rows) {
             System.out.format("%5d", col);
         }
             System.out.println();
         }
        return v[rows1][capacity];
    }
    public static void main(String[] args) {
        int[] v={10,40,30,50};
        int[] w={5, 4, 6, 3};
        int c=10;
        System.out.println(knapsack(v,w,c));
    }

}
