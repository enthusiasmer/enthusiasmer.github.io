单链表的反转

发表于 2016-11-11   |   分类于 数据结构——链表   |  

思路分析

将单链表进行反转，博主用了两种方法来解决此问题；
1.利用递归，进行递归反转，即从链表尾部一次转换结点指向；
2.利用遍历进行反转，从链表头进行反转，反转前需要把当前结点的next指针进行保存。

实现代码

package Algorithms;
public class Solution206 {
    //方法一：递归反转法
    public static ListNode reverseList1(ListNode head){
        if(head==null||head.next==null) return head;//如何为空链表或者head为尾结点，直接返回
        ListNode start=reverseList1(head.next);
        head.next.next=head;
        head.next=null;
        return start;
    }
    //方法二：遍历反转法
    public static ListNode reverseList2(ListNode head){
        if(head==null||head.next==null) return head;
        ListNode pre=head;//定义为当前结点的前结点
        ListNode cur=head.next;//定义cur为当前结点
        ListNode temp=null;//临时结点用来放置当前结点的next
        while(cur!=null){
            temp=cur.next;
            cur.next=pre;
            pre=cur;
            cur=temp;
        }
        head.next=null;
        return pre;
    }
    public static void main(String[] args) {
        ListNode lis1=new ListNode(1);
        ListNode lis2=new ListNode(2);
        ListNode lis3=new ListNode(3);
        ListNode lis4=new ListNode(4);
        lis1.next=lis2;
        lis2.next=lis3;
        lis3.next=lis4;
        //反转前，打印链表
        ListNode h=lis1;
        while(h!=null){
            System.out.print(h.val+" ");
            h=h.next;
        }
        //反转后，打印链表
        System.out.println("\n======================");
        ListNode eh=reverseList2(lis1);
        while(eh!=null){
            System.out.print(eh.val+" ");
            eh=eh.next;
        }
    }
}
